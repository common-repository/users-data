== Description ==

This plugin mainly helps the admin to get thier users data such as user id, user's display name, user's email i'ds. 

This will help admin to get a clear idea of how many users are there in the website.

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Users Data` and click 'Install Now' or click on the `upload` link to upload `users-data.zip`
4. Click on `Activate plugin`
